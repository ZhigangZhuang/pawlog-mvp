# Design

当前改用项目内本地 UI Board 管理设计，不依赖 Figma 登录。

打开：

```text
design/ui-board.html
```

内容：

- 产品原则
- Design Tokens
- 首页 Feed
- 动态详情
- 毛孩主页
- 图鉴
- 发动态
- 地图
- 我的

使用方式：

- 产品方向写入 `docs/PRD.md`
- 交互红线写入 `docs/UX_RULES.md`
- 页面视觉基准看 `design/ui-board.html`
- 当前真实界面截图看 `design/screenshots/`
- 每次大改动写入 `docs/CHANGELOG.md`
