## Update Rule

大改 UI 后重新截图，并检查：

- 首页仍然是动态流，不出现大功能入口
- 图鉴承担搜索和筛选
- 动态详情展示关联毛孩
- 毛孩主页优先展示动态和轻量关系
- 分享关系统一使用“我的 / 分享给我的 / 允许我记录 / 只读”
- 流浪动物位置仍然默认模糊
