# Changelog

## 2026-05-28

### 多动物发布验收修复

- 发动态保存后会回到主角毛孩主页
- 多动物动态同时出现在所有关联毛孩主页
- 发布时为每条动态生成独立 ID，避免复用旧动态 ID
- 添加面板动作会先关闭面板再跳转，避免残留遮挡
- 发动态页面底部留白加大，避免发布按钮被底部导航误触

影响页面：

- 发动态
- 首页
- 毛孩主页
- 添加入口

破坏性变更：

- 无

- 动态详情

- 添加入口

- 无。该改动为结构拆分，页面行为保持不变。

## 2026-05-28

### 多动物动态与关系网

- Post 支持关联多个 Animal
- 增加 `primary_animal_id` 和 `animal_ids`
- 增加 `PostAnimal`
- 增加 `AnimalRelationship`
- 增加 `AnimalFamily`
- 增加 `AnimalTransfer`
- 发动态支持选择多只毛孩并设置主角
- 首页支持多动物动态标题
- 动态详情展示“这条动态里的毛孩”
- 动物主页展示所有关联该动物的动态
- 动物档案 Tab 展示关系和送养信息
- 增加猫妈妈一家三口、奶盖小猫送养、豆包和 Lucky 的 mock 场景

影响页面：

- 首页
- 动态详情

- 我的
